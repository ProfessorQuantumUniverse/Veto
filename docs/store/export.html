Veto store asset exporter

Serve this folder over HTTP so the browser may read the SVG sources, then open this page:
cd docs/store && python -m http.server 8000 then visit http://localhost:8000/export.html. Click Download all to save every asset as PNG at its exact target resolution, or download them one by one. The screenshots land at 1080 by 2160 and the feature graphic at 1024 by 500.
